Web Clip Leads from GitHub

Easily export lead data from GitHub to a CSV file.

How to Use

  1. Navigate to a GitHub repository or user profile you want to web clip leads from.
  2. Click the extension icon next to the repo name to save it to your list.
  3. Review the lead data that has been automatically web clipped.
  4. Click the "Export to CSV" button to download the lead data.
